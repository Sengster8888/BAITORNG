## Example Working Node.js / JavaScript Usage

```js
const provinceData = require('./provinceNeighbors.json');

function getNearestProvinces(currentProvince, limit = 10) {
  const neighbors = provinceData[currentProvince];

  if (!neighbors) {
    return {
      success: false,
      message: 'Province not found'
    };
  }

  return {
    success: true,
    currentProvince,
    nearest: neighbors.slice(0, limit)
  };
}

// Example
const result = getNearestProvinces('Phnom Penh');

console.log(result);
```

## Why This Structure Works

- Fast lookup → O(1)
- No database query needed
- Very lightweight
- Easy to maintain
- Perfect for MVP
- Easy to migrate later to MySQL/PostGIS

## Important Note

These distances are approximate MVP-level centroid distances, not exact road distances.

For production:
- Use real GIS coordinates
- Calculate automatically with the Haversine formula
- Or use Google Maps Distance Matrix API for real travel distance

But for your MVP and university project, this structure is already very solid.
